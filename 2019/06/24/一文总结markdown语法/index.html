一文总结markdown语法

Posted on 2019-06-24 |

简述

由于markdown语法简洁、优雅、跨平台、兼容html等优点，出于写博客需要，现总结markdown语法，推荐使用在线markdown编辑器或visualstudiocode配合markdown插件，可实时预览效果。故本文现学现用，使用markdown总结markdown语法:)

语法

包括常用语法和扩展语法。常用语法是标准语法，扩展语法需要编辑器支持。

换行

两空格在加文字。

标题

setext形式

上面是标题，换行加上=或-符号，但只支持标题1和标题2。

atx形式

对称#包括文本

用对称的 # 包裹文本，即文本前后使用数量相等的#包裹文本。

仅左边使用#

左侧也可用标题数个数的’#’+空格+文字，最多六级标题。

列表

无序列表

符号(、+或-)+空格。
* 符号作为标记。
+ 也可用+。
- 还有-。

  • 符号*作为标记。
  • 也可用+。
  • 还有-。

有序列表

数字+.+空格

1. 列表1。
2. 列表2。

  1. 列表1。
  2. 列表2。

嵌套列表

  • 1
    • 1

分割线

一行或多行使用三个以上*、-、_表示分割或，符号之间仅可有空格。
*********
我是分割线:)
*********

我是分割线:)

引用

段落或句前加>符号，多行引用可每行前加>或仅段首1个>符号，若引用要换行可再行尾加两空格或>+换行。
>引用

引用

强调

斜体

*邪体* 或 _斜体_或<em>邪体</em>。

邪体 或 _斜体_或邪体。

加粗

**加粗**或 __加粗__ 或<strong>加粗</strong>。

加粗或 加粗 或加粗。

超链接

行内式

按照[link text](URL)。
[百度](https://www.baidu.com)

百度

参考式

等价于将行内式分成两部分。

自动链接

http://www.baidu.com、123@qq.com。

图像

和超链接一致，但在前加!，另可通过<img/>指定图像大小。

代码

代码前后至少一个空行，且每行前需要至少一个Tab或四空格。也可通过 ``，插入行内代码（` 是 Tab 键上边、数字 1 键左侧的那个按键）。

字符转义

反斜杠+特殊字符（\、`、*、_、{}、[]、()、#、+、-、.、!）

扩展语法

由于很多编辑器不支持或部分支持扩展语法，建议尽量使用标准语法，基本上够用。

删除线

在待删除语句左右加上两个~。
~~删除~~

删除

代码块

用两个```包裹代码块。
```
代码
```

注意```要独占一行。

代码高亮

在第一组后面加上代码所属语言即可。

```js
window.addEventListener(‘load’, function() {
console.log(‘window loaded’);
});
```

表格

|分割不同单元格而用-分隔表头和其他行。左对齐:--右对齐--:居中对齐:--:。

| left | center | right |
| :-- | :–: | --: |
| a | b| c|

left center right
a b c
