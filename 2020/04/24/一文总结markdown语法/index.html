一文总结markdown语法

Posted on 2020-04-24

由于markdown语法简洁、优雅、跨平台等优点，出于写博客需要,现总结markdown语法，本文现学现用，使用markdown总结markdown语法:)

  1. 换行
    两空格或<br>
    1. 标题
      #+空格+文字，最多六级标题
  2. 列表
  • 无序列表
    符号(*、+或-)+空格
  • 有序列表
  1. 数字+.+空格
  • 嵌套列表
    • 1
    • 2
    1. 3
  1. 分割线

一行或多行使用三个以上*、-、_表示分割或<hr>，符号之间仅可有空格

5. 引用

段落或句前加>符号，多行引用可每行前加>或仅段首1个>符号，若引用要换行可再行尾加两空格或>+换行
6. 强调

  • 斜体: 邪体 或 _斜体_或邪体
  • 加粗:加粗或 加粗 或加粗
  1. 超链接
  • 行内式　link text
  • 参考式:等价于将行内式分成两部分
    link text
  • 自动链接 http://www.baidu.com、123@qq.com
  1. 图像
    和超链接一致，但在前加!，另可通过指定图像大小
  2. 代码块
    代码块前后至少一个空行，且每行前需要至少一个Tab或四空格。也可以通过 ``，插入行内代码（` 是 Tab 键上边、数字 1 键左侧的那个按键）
  3. 字符转义
    反斜杠+特殊字符（\
    `
    *
    _
    {}
    []
    ()
    #
    +
    -
    .
    !）
  4. 扩展语法
    1. 删除线:删除
    2. 代码块: ```要独占一行
    3. 代码高亮:在第一组后面加上代码所属语言即可
    4. 表格:|分割不同单元格而用-分隔表头和其他行。左对齐:–右对齐–:居中对齐:–:
    left center right
    a b c
    5. task list
  • [ ] Eat
  • [ x ] Code
    • [ x ]
  1. 支持linux的markdown编辑器
    CuteMarkEd、Haroopad、ReText、Cmd Markdown、StackEdit(web)、Dillinger(web)

推荐使用在线markdown编辑器或vs code配合markdown插件，可实时预览效果。
参考自Learning-Markdown (Markdown 入门参考)
